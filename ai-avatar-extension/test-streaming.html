AI Avatar Chrome Extension - Streaming Test Page

This page is designed to test the streaming functionality of the AI Avatar Chrome Extension. Open the extension popup while viewing this page to test the new streaming features.

New Streaming Features

Real-time Response Streaming

Messages from the AI now stream in real-time, providing a more interactive experience. You'll see the text appear word by word as the AI generates its response.

Visual Streaming Indicators

The extension now shows animated wave indicators when the AI is typing, replacing the static "thinking" indicator during the streaming phase.

Abort Functionality

You can now stop a streaming response mid-generation by clicking the red abort button that appears during streaming.

Avatar State Updates

The 3D avatar transitions to a "speaking" state during streaming, providing visual feedback that matches the streaming text.

Settings Toggle

Streaming can be enabled or disabled in the settings panel. When disabled, responses will appear all at once like before.

How to Test

1. Basic Streaming Test

  • Open the extension popup
  • Ask a question about this page
  • Watch as the response streams in real-time
  • Notice the animated wave indicator and avatar state

2. Abort Test

  • Start a new conversation
  • While the AI is streaming a response, click the red abort button
  • The response should stop immediately with an "(Response aborted)" note

3. Toggle Streaming

  • Go to Settings in the extension
  • Uncheck "Enable streaming responses"
  • Send a message - it should appear all at once
  • Re-enable streaming and test again

4. Error Handling

  • Test with an invalid API key to see error handling
  • Test rate limit handling during streaming
  • Test network interruption scenarios

Technical Implementation

The streaming implementation uses:

  • Server-Sent Events (SSE) for real-time data streaming
  • AbortController for canceling ongoing streams
  • Chrome Runtime Messages for communication between service worker and popup
  • CSS animations for visual feedback
  • Three.js animations for avatar state changes
Note: Make sure you have configured your OpenAI API key in the extension settings before testing. The streaming feature requires a valid API key to function.
